## Como rodar

#### 1 - Instala as dependencias

    npm install
    

#### 2 - Rodar aplicação
    
    npm run build
    npm start

#### 3 - Visualizar
  - Abre o navegador e busque por [local](http://localhost:3000/)
  - ou busque por [site](https://gustavo-3000.code.fslab.dev)
